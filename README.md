# Key
Key é um aplicativo de gerenciamento de senhas multiplataforma

Em desenvolvimento sem todas as funcionalidades ainda implementadas.

Principais requerimento:
* Multiplataforma Linux, Windows e Android.
* Designer responsivo 
* Banco de dados portável 
* Minimalista
* Sistema de criptografia de dados
* Conexão com Ondrive :)
* Sistema de login
* Trabalhar com múltiplos bancos de dados
* Trabalhar com múltiplos usuários 

Diretivas:
* A princípio o sistema será escrito na IDE Lazarus voltado a obtenção da versão desktop para windows e linux
* Versão multiplataforma para máquina java pode ser desenvolvido paralelamente
* Versão mobile poderá ser desenvolvido em flutter ou java

Requerimento para o desenvolvimento da versão Desktop:
* IDE Lazarus instalado
* Componente zeos instalado
